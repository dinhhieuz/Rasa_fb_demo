Hiện trạng:¶

Hiện tại em đang muốn thực hiện 1 đoạn code Python bên trong SQL bằng SQL Machine Learning được hỗ trợ từ SQL.

Code Python mà em muốn gữi đi là 1 cái Request API tới 1 URL.

Vấn đề:¶

Khi chạy thì Em không thể gữi đi được request API (Mọi code khác thì được). Em có thử trên Jupter notebook thì lại không vấn đề gì ạ, nhưng khi chạy trong SQL thì lại lỗi

Code SQL:¶

    , @script = N'

response = requests.get(url = "https://6237d6d9f5f6e28a154f3fa2.mockapi.io/add")
